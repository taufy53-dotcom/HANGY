# 🏰 HANGY

A medieval-themed Hangman game built with **Python**, **CustomTkinter**, and **Pygame**.

Fight your way through challenging words, earn score, rank up, and become the ultimate word master!

---


## ✨ Features

- 🏰 Medieval-themed interface
- 🎵 Background music and sound effects
- 🧩 Multiple word lengths
  - 2 Letters
  - 3 Letters
  - 4 Letters
  - 5 Letters
  - 6 Letters
- 💡 Hint system
- 🏆 High Score tracking
- ⭐ Player Rank system
- 👤 Player Profile
- 💾 Automatic save system using SQLite
- 🖥️ Modern UI built with CustomTkinter
- 🎮 Easy to play

---

## 🛠️ Built With

- Python 3.12+
- CustomTkinter
- Pillow
- Pygame
- SQLite3

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/taufy53-dotcom/HANGY.git
```

Go into the project

```bash
cd HANGY
```

Install dependencies

```bash
pip install customtkinter pillow pygame
```

Run the game

```bash
python main.py
```

---

## 🎮 Gameplay

- Choose a word length.
- Guess letters one by one.
- Avoid making too many wrong guesses.
- Use hints wisely.
- Earn score.
- Increase your rank.
- Beat your highest score.

---

## 💡 Hint System

| Word Length | Hints |
|-------------|-------|
| 2 Letters | 0 |
| 3 Letters | 1 |
| 4 Letters | 1 |
